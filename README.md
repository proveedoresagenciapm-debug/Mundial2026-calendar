## 🚀 Cómo publicarlo en GitHub (paso a paso)

### PASO 1 — Crea tu cuenta GitHub
1. Ve a **https://github.com**
2. Haz clic en **"Sign up"**
3. Completa el registro con tu email

---

### PASO 2 — Crea un repositorio
1. Una vez dentro, haz clic en el botón verde **"New"** (o el ícono `+` arriba a la derecha)
2. En **"Repository name"** escribe: `mundial2026-calendar`
3. Selecciona **"Public"** ← ⚠️ MUY IMPORTANTE: debe ser público
4. Marca la casilla **"Add a README file"**
5. Haz clic en **"Create repository"**

---

### PASO 3 — Sube los archivos
1. Dentro del repositorio, haz clic en **"Add file"** → **"Upload files"**
2. Arrastra y suelta los 3 archivos:
   - `mundial2026.ics`
   - `index.html`
   - (reemplaza el README.md existente con este)
3. Haz clic en **"Commit changes"**

---

### PASO 4 — Activa GitHub Pages (tu sitio web gratis)
1. En tu repositorio, haz clic en **"Settings"** (pestaña superior)
2. En el menú izquierdo, haz clic en **"Pages"**
3. En **"Source"**, selecciona **"Deploy from a branch"**
4. En **"Branch"**, selecciona **"main"** y carpeta **"/ (root)"**
5. Haz clic en **"Save"**
6. Espera 2-3 minutos y tu sitio estará en:
   ```
   https://TU_USUARIO.github.io/mundial2026-calendar/
   ```

---

### PASO 5 — Obtén tu URL del archivo .ics
Esta es la URL que necesitas para los calendarios:
```
https://raw.githubusercontent.com/TU_USUARIO/mundial2026-calendar/main/mundial2026.ics
```

⚠️ **Reemplaza `TU_USUARIO`** con tu nombre de usuario de GitHub.

**Ejemplo real:**
```
https://raw.githubusercontent.com/pasaportemagico/mundial2026-calendar/main/mundial2026.ics
```

---

### PASO 6 — Actualiza los links en index.html
Abre `index.html` y busca esta línea al final:
```javascript
const ICS_URL = "TU_URL_AQUI";
```
Reemplázala con tu URL real:
```javascript
const ICS_URL = "https://raw.githubusercontent.com/TU_USUARIO/mundial2026-calendar/main/mundial2026.ics";
```

Luego sube el archivo actualizado al repositorio.
